Practice Questions


1.What is the difference between shutil.copy() and shutil.copytree()?

A.

	shutil.copy() - copy a single file

	shutil.copytree() - copy an entire folder along with all its contents.


2.What function is used to rename files?

A.

	shutil.move()


3.What is the difference between the delete functions in the send2trash and shutil modules?

A.

	delete functions in send2trash will move to recycle but the delete functions in shutil modules delete the files permanently.


4.ZipFile objects have a close() method just like File objects’ close() method. What ZipFile method is equivalent to File objects’ open() method?

A.

	zipfile.zipFile is eqivalent to open() function.
